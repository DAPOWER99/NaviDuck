Get Started in 60 Seconds

One file. No installation. Just works.

1

Download NaviDuck

# One command to get everything
curl -sSL https://raw.githubusercontent.com/DAPOWER99/NaviDuck/main/naviduck.py -o naviduck.py

# Or clone the repository
git clone https://github.com/DAPOWER99/NaviDuck.git
cd NaviDuck

That's it. No package managers, no dependencies (unless you want optional features).

2

Run It

# Basic run (works anywhere)
python naviduck.py

# Or if you have Python 3 specifically
python3 naviduck.py

# Make it executable and run directly
chmod +x naviduck.py
./naviduck.py
3

Optional: Enhanced Features

# Install requests for better performance
pip install requests

# Enable Tor support (requires Tor Browser)
# Set tor_enabled = True in settings

These are optional. NaviDuck works perfectly without them.
